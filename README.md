## 학습 목표

- 바이브 코딩으로 데이터 대시보드를 직접 제작할 수 있다
- AI를 활용한 업무 자동화 워크플로우를 설계하고 구현할 수 있다
- 외부 서비스(Gmail, Slack 등)와 API 연동을 할 수 있다
- 에러 메시지를 보고 스스로 문제를 해결할 수 있다

## 기술 스택

- **정적 사이트**: Jekyll + GitHub Pages (Cayman 테마)
- **콘텐츠**: Markdown (.md)
- **실습 도구**: Claude, Opal, Cloudflare, Apps Script, AppSheet, n8n
